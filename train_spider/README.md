# 12306车票查询
 
练习面相对象编程！
 
运行方法：在命令行输入python3 run.py 

普通查询方法：出发站 到达站 年月日

按车型查询方法：车型代码 出发站 到达站 年月日

车型代码对照：G-高铁，C-城际，D-动车，Z-直达，T-特快，K-快速
